## цель работы
закрепить навыки объектно-ориентированного программирования, работы со стандартной библиотекой stl, файлового ввода-вывода, проектирования модульных приложений и обработки пользовательского ввода в среде c++.

## задача
разработать консольное приложение для автоматизации учёта товаров на складе. программа обеспечивает добавление, редактирование, удаление записей, поиск и фильтрацию, сортировку, расчёт аналитических показателей, а также сохранение и загрузку данных в текстовом формате txt.

## технические требования
- язык программирования: c++17
- хранение данных: std::vector
- формат файла: txt с разделителем запятая
- обработка ошибок ввода без аварийного завершения
- стиль кода: google c++ style guide
